# World Wide Importerts Sales Dashboard

1: Overview:

This project was to design a dynamic, interactive tool to explore world importers sales and costs. To be able to determine their profitability vs product costs.

2: Details:
This gives us insights into the the world importers from Australia, Canada, France, Germany, United Kingdom, and Unites States.
We can drill down to each country to see their total sales vs product cost
We can drill further down to see each individual product cost
We can also compare the sales amount per country

3: Tech Stack
The dashboard was using the following tools and technologies
-- 1: Power BI Desktop 
      -- Main data visualization platform used for report creation.
-- 2 Power Query
      -- Data transformation and cleaning layer for reshaping and preparing data
-- 3: Data Modeling
      -- Relationships established between tables.

5: Features / Highlights
      -- Business Problem
            -- Global importers generate millions in revenue, yet they need insight into specifics to determine where growth opportunities are and what products are the most profitable

      -- Questions 
            -- Which country has the highest sales?
            -- Which country has the lowest sales?
            -- Which product is the most profitable?
            -- Which producct is the least profitable?

6: Visualizations used
      -- Clustered Column Chart
            -- This provis the total sales amounts vs total product cost 
      -- Slicer 1
            -- Calculates the total sales amounts
            -- This can drill down to each individual country
      -- Slicer 2
            -- THis provides the countries to filter by
      -- Product Table
            -- This allows to a view of each specific product and their cost
